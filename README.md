Cinema Ticket Booking System

Overview

This project is a simple Cinema Ticket Booking System developed in C# to demonstrate the practical application of Object-Oriented Programming (OOP), SOLID principles, and Software Design Patterns. The project focuses on writing clean, maintainable, and extensible code rather than building a complete commercial cinema system.

Project Features

Register different ticket types (Regular, Student).
Calculate ticket prices based on ticket type and discounts.
Confirm bookings and send email notifications.
Prevent double-booking of the same seat.
Simple and organized user workflow.

Project Purpose

The purpose of this project is not to build a real cinema booking platform. Instead, it demonstrates the ability to design software using proper architecture and design principles.

The project shows how to:

Apply suitable Design Patterns to solve real programming problems.
Follow SOLID principles to produce clean and maintainable code.
Build software that is easy to extend without modifying existing functionality.
Organize responsibilities between classes using good object-oriented design.
